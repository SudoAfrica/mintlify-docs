## Common runtime values

These values appear across multiple requests:

- `institutionId`
- `institutionToken`
- `oauthToken`
- `oauthRefreshToken`
- `cardId`
- `institutionUserId`
- `cardProgramId`
- `apiKey`

## How to use them

Store these values in your API client or Postman environment so you can reuse them across requests without editing each payload manually.
