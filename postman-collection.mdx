## Overview


## Suggested variables

Create these collection variables before you run requests:

- `scheme`
- `baseUrl`
- `institutionId`
- `institutionToken`
- `oauthToken`
- `apiKey`

For sandbox testing, set `scheme` to `https` and `baseUrl` to `sandbox-api.cardcore.cloud`.
## Recommended first requests

Start with:

- `List OAuth Applications`
- `Create OAuth Application`
- `OAuth Token`
- `Get Banks`
- `Name Enquiry`
