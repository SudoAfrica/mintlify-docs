## Cardcore Reference

The API reference pages in this section provide interactive request and response details for the available Cardcore endpoints.

## Live Calls

You can send requests directly from each endpoint page once you set:

- The required credentials for the sandbox host `https://sandbox-api.cardcore.cloud`.
- A bearer token for institution-protected routes.
- An API key for the `Import Card` utility endpoint.

## Coverage

Most available endpoints are documented here as live playground pages. A small number of flows still require confirmed route details and are called out in the guides as coverage gaps.
